   Add Vehicle Masters

Top

    To Vehicle Masters

  1. Select the Vehicle Type

    2. Eg : Suppose the Vehicle Type is "Car".
    you have to select Car for "Vehicle Type".
  2. Enter the Vehicle Number

    3. Eg : Suppose the Vehicle Number is "AP07AV8288".
    you have to enter AP07AV8288 for "Vehicle Number".
  3. Pick the Purchase Date
  4. Enter the Purchase Cost
  5. Select the Unit

    6. Eg : Suppose Unit is "Head Office".
    you have to Select Head Office for "Unit".
  6. Enter the RC Number
  7. Enter the Insurance Number
  8. Pick the Insurance expiry date
  9. Enter the capacity
  10. Enter the remarks,if any
  11. Browse for image
  12. Once filled click on 'save' button to save Vehicle Masters details or 'CANCEL' to exit without saving the Vehicle Masters details

 To Add Vehicle Trip Details

Top

  1. You have to select vehicle type whether it is car or lorry or bus etc...
    Eg :Suppose if it is "Car" then, you have to select Car as "Vehicle Type".
  2. You have to select Vehicle Number.(the unique no assigned to that vehicle).
    Eg :Suppose if it is "46061" then you have to select46061 as "Vehicle Number".
  3. you have to select driver of vehicle.
    Eg :Suppose if driver is "Azim" then, you have to select Azim as "Driver".
  4. You have to select start date.
    Eg :Suppose start date is "05.01.2012" then, you have to select 05.01.2012 for " Start Date".
  5. You have to enter start time.
    Eg :Suppose start time is "07.30" then, you have to enter07.30 for " Start Time".
  6. You have to select start Place.
    Eg :Suppose start place is "Bengalore" then, you have to select Bengalore for " Start Place".
  7. You have to enter start reading.
    Eg :Suppose if starting reading is "4587" then, you have to enter4587 as "Start Reading".
  8. You have to select end date.
    Eg :Suppose end date is "05.01.2012" then, you have to select 05.01.2012 for " End Date".
  9. You have to enter end time.
    Eg :Suppose end time is "07.30" then, you have to enter07.30 for " End Time".
  10. You have to select End Place.
    Eg :Suppose End place is "Mysore" then, you have to select Mysore for " End Place".
  11. You have to enter End reading.
    Eg :Suppose if end reading is "4882" then, you have to enter4882 as "End Reading".
  12. You have to enter Expences Incurred.
    Eg :Suppose if expences are "3000 Rupees" then, you have to enter3000 as "Expences Incurred".
  13. You have to select whether the vehicle is travelled with load or without load.
    Eg :Suppose it is "With load" then, you have to select With load.
  14. If you want to write something about Renewal write it down as Remarks.
  15. Once filled click "SAVE" button to Save "Vehicle Trip" or click "CANCEL" button to exit without Saving "Vehicle Trip".
