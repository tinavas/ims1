 Charge Alerts Report

Top

     You have to select Charge Alerts in Reports.
     As and when you select Charge Alerts, a new window will be opened.
     The Report shows vehicle Unit, Type, Number,Charge Code and Renewal Date fields and its values.

  1. You can select single or all Vehicle Types/Vehicle Numbers/Unit at a time using Vehicle Type/Vehicle Numbers/Unit drop down list box.
      2. Based on the selected Vehicle Type/Vehicle Number/Unit corresponding details will be shown to you.

 Service Details Report

Top

     You have to select Service Masters in Reports.
     As and when you select Service Masters, a new window will be opened.
     The Report shows Date, vehicle Type, Number, Service Code, Service Description,Tentative Next Service Date,Warranty,Service Charges,Driver,Narration fields and its values.

  1. You can select single or all Vehicle Types/Service Code/Service Description at a time using Vehicle Type/Service Code/Service Description drop down list box.
      2. Based on the selected Vehicle Type/Service Code/Service Description corresponding details will be shown to you.

 Trip Details Report

Top

     You have to select Trip Details in Reports.
     As and when you select Trip Details, a new window will be opened.
     The Report shows Trip Startdate , Start Place , Vehicle Type, Vehicle Number, Start Time, Start Reading , End Place , End Date , End Time, End Reading, Driver, Work Load and Expenses Incurredfields and its values.

  1. You can select individual or all Startdate/Startplace/Vehicle Type/Vehicle Number/Start Time/Start Reading/End Place at a time using the Startdate/Startplace/Vehicle Type/Vehicle Number/Start Time/Start Reading/End Place drop down list box.
      2. Depending on the options you select, report will display corresponding values.

  1. You can select single or all Vehicle Unit/Type/Number at a time using Vehicle Unit/Type/Number drop down list box.
