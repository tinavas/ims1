Masters

   Add Quail Rates Record

Top

    To Add Quail Rates Record

  1. You have to select "Date" and "unit".


  2. Now select "Category" , "Code" and " Description"
    If you select the 'Code' you will get the 'Description' and if you select the 'Description' you will get the 'code'

    3. Eg : Suppose the Date is "6th June, 2011",unit is " Godown",Category "Processing Unit", Code "QUAIL101" , Description "Quail Processed", Price "23".

    you have to select 06.06.2011 for "Date",Godown for "Unit ", Quail for "Category " QUAIL101 for "Code",Quail Processed for "Description",enter 23 for "Price".

  3. Repeat the above 2nd step until all the details are filled.


  4. Once filled click on 'SAVE' button to save the Quail rate details or 'CANCEL' to exit without saving the Quail Rate details.

Transactions

   Add Retail Rates Record

Top

    To Add Retail Rates Record

  1. You have to select "Date" , "Cash Code" and "Unit".


  2. Now select "Category" , "Code" and " Description"
    If you select the 'Code' you will get the 'Description' and if you select the 'Description' you will get the 'code'

  3. Enter the values for " Quantity " and " Price "

    4. Eg : Suppose the Date is "30th September, 2011",cashcode is "AS103",unit is " Godown",Category "Quail", Code "QUAIL101" , Description "Quail Processed", Quantity "35",Price "23".

    you have to select 30.10.2011 for "Date",AS103 for "Cashcode ",Godown for "Unit ", Quail for "Category " QUAIL101 for "Code",Quail Processed for "Description",enter 35 for "Quantity" and 23 for "Price".

  4. Repeat the above 2nd and 3rd steps until all the details are filled.


  5. Enter the Discount value and Cleaning Charges, then the Net Amount will be displayed automatically.
  6. Once filled click on 'SAVE' button to save the Quail Retail Sales details or 'CANCEL' to exit without saving the Quail Retail Sales details.

  3. Select "Category","Code","Description" and "To"(To Warehouse).

  4. Enter values for "Sentquantity","Mortality" then you will get the "Received Quantity" automatically.
    Enter "Price";

    5. Eg : Suppose the Live Birds Transfer Date is "27th march, 2012",if you want to transfer an item from "Processing Unit",DC is "111", T.Cost is "1500", Vehicle Number is "KA072331", Driver is "Azim", Category is "Quail Birds", Code is "QUAIL101", Description is "Quail Female Birds", To "Chicken Shop", Sent Quantity is "800", Mortality is "10" and Price is "80"

 Quails Processing

Top

Add Quails Processing

  1. You have to select date.
  2. You have to select Processing Unit.
    Eg :Suppose the Processing Unit is "Processing Unit" you have to select Processing Unit for "Processing Unit".
  3. You have to enter Quails Age.
  4. You have to Enter Quails Processed and Rejected Items.Then we will get Finished Quails automatically
    Eg :Suppose the Quails Processed is "1500" and Rejected Items is "200" you have to enter 1500 for "Quails Processed" and 200 for "Rejected Items".
  5. You have to select PR.Quail Code (Processed Quail Code) and PR.Quail Description (Processed Quail Description).
    Eg :Suppose the PR.Quail Code is "PQL101" you have to select PQL101 for "PR.Quail Code" and PQL102 for "PR.Quail Description".
  6. You have to enter Packets.
    Eg :Suppose the Packets is "10" you have to enter 10 for "packets".
  7. Repeat the steps 5 and 6 until all the details are filled
  8. The Total Amount and Fresh Processed Quails will be displayed automatically.
  9. If any Narration write in narration textbox
  10. Once filled click "SAVE" button to Save the Birds Transfer or click "CANCEL" button to exit without Saving the Birds Transfer.

 Birds Receiving

Top

Add Birds Received

  1. You have to select date.
  2. You have to select Received By(the employee who received birds).
    Eg :Suppose if the employee is "Steven" you have to select Steven for "Received By".
  3. You have to select Processing Unit(from where you received birds).
    Eg :Suppose if the Processing Unit is "Puttur" you have to select Puttur for "Processing Unit".
  4. You have to select Type(Either Fattener Birds or Quail Birds).
    Eg :Suppose if the Type is "Quail Birds" you have to select Quail Birds for "Type".
  5. You have to select DcNumber.
    Eg :Suppose if the DcNumber is "1562" you have to select 1562 for "DcNo".
  6. You have to enter Mortality.
    Eg :Suppose if Mortality is "120" you have to enter 120 for "Mortality".
  7. You have to enter Birds Received.
    Eg :Suppose if Received Birds are "600" you have to enter 600 for "Received Birds".
  8. If you want to write anything about birds receiving write it in narration box.
  9. Once filled click "SAVE" button to Save the Birds Reveive or click "CANCEL" button to exit without Saving the Birds Receive.
