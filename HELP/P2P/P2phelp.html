    3. Eg: If suppose the Name of Supplier is "Obafemi Trading Co.", Address is "Lagos", Phone Number is "08042111111", Mobile Number is "09010203040", Contact Type is "Vendor",Credit Terma is "2 days" and Supplier Group is "00458" you have to enter Obafemi Trading Co. for "NAME" ,Lagos for "ADDRESS",08042111111 for "PHONE" and 09010203040 for "MOBILE".

Add New Tax Master

  1. You have to enter the values of "Tax Code" and "Description"
  2. You have to select the value of "Deductions" , if it is "Charges" you have to select the value of "Charge Type" also.
  3. Now select the value for "Rule",you have to selct the value of "CoA".
  4. Select either "Percent" or "Flat" by selecting the appropriate radio button and enter the values of "Tax Code Value".
  5. You have to select the value of "Applicable".
    6. Eg :Suppose Tax Code is "tax001", Description is "taxcode001", Deductions is "Charges", Charge Type is "Freight", Rule is "Exclude", CoA is "IT401", Tax Code Value is "1.5" and Pplicable is "Purchases" then,
    you have to enter tax001 for "Tax Code", taxcode001 for "Description", select Charges for "Deductions", Freight for "Charge Type", Exclude for "Rule", IT401 for "CoA".
    Now enter 1.5 for "Tax Code Value" and "Purchases for "Applicable".

   Add New Purchase Indent

    To Add new Purchase Indent
  1. You have to select the value of "Date".
  2. select values for "Category" and "Item Code" from dropdowns provided.
    The "Description" and "Units" will be displayed automatically as and when the value of Item Code is selected.
  3. Now enter the the value of "Quantity" requested.
  4. Select values for "Req. Date", "Delivery Office", "Receiver".

    5. Eg: If suppose the Purchase Indent Date is "03.01.2011", Category is "Ingredients", Item Code is "MZ101", Quantity is "100", Delivery Office is "Banglore office" and Receiver is "Ganesh" you have to select 03.01.2011 for "REQ. DATE" ,Ingredients for "CATEGORY" and MZ101 for "ITEM CODE".
    The value of "DESCRIPTION" is automatically displayed as Maize and "UNITS" will be displayed as Kgs as and when the Item Code is selected.
    Now enter 100 for "QUANTITY" ,select Banglore office for "DELIVERY OFFICE" and Ganesh for "RECEIVER" .
  5. Repeat the above steps 2,3 and 4 until required details are filled.
  6. Once filled click on 'SAVE' button to save the Purchase Indent or 'CANCEL' to exit with out saving the Purchase Indent.

   Add New Purchase Order

    To Add new Purchase Order
  1. First You have select "With Purchase Request" or With Out Purchase Request"
  2. Select values for "Vendor", "Date"(Purchase Order Date).
    Eg : Suppose Vendor is "A B M Industries", Purchase Order Date is "31.12.2010"
    you have to select A B M Industries for "Vendor" and 31.12.2010 for "Date" .
  3. You have to select the values of "Category", "Item Code", "Delivery Date" and "PR #" if "With Purchase Request".
    Enter the values of "Rate/Unit" and "Quantity" if "With Out Purchase Request".
    Eg :Suppose Category is "Ingredients", Item code is "MZ101", PR # is "PR-1210-0004", Rate/Unit is "12" and Delivery Date is "05.01.2011" you have to select Ingredients for "Category", MZ101 for "Item Code", PR-1210-0004 for "PR #", 05.01.2011 for "Delivery Date" and enter 12 for "Rate/Unit".

    4. Click on Delivery Details

  4. You have to select "D.Location"(Delivery Location) and "Receiver" from dropdowns.
    Eg :Suppose Delivery Location is "banglore Office" and Receiver is "Ganesh" you have to select Banglore Office for "D. Location" and Ganesh for "Receiver".
  5. You can select "Tax" select the dropdown provided.
    Eg :Suppose if Tax is "VAT1P" you have to select VAT1P for "Tax".
  6. You can Select "Frieght" select the dropdown provided.
    Eg : Suppose Frieght is "FR001" you have to select FR001 for "Frieght".
  7. Select "Discount" from the dropdown provided.
    Eg : Suppose Discount is "SD001" you have to select SD001 for "Discount" from the dropdownlist.
  8. Select appropriate values of Terms and Conditions from the select box provided.
    As and when the value is selected the description will be displayed automatically.
    Eg : Suppose "Frieght 50" and "CFORM" are the Terms & Conditions then you have to select both Frieght 50 and CFORM from Select Box then the Description Such as 50 % Freight will be given will be displayed automatically.
  9. Once filled click on 'SAVE' button to save the Purchase Order or 'CANCEL' to exit without saving.

   Add New Direct Purchase

    To Add new Direct Purchase
  1. You have to select the "Date" of the Purchase, "Vendor", and enter the value of "Book Invoice"(if not available make it "0"(Zero) ) .
    The "Invoice" value will be generated automatically based on which Payments will be made

    2. Eg: If suppose for the Invoice "SOBI-0111-0020", Purchase Date is "05.01.2011" and Vendor is "ALTECH" you have to select 05.01.2011 for "Date" and ALTECH for "Vendor".
    Note that the Invoice Number SOBI-0111-0020 will be generated automatically.
  2. You can select "Category" and "Code".
    As and when the "Code" value is selected the "Description" and "Units" of the selected item will be displayed automatically.
  3. Now enter the "Quantity Sent"(By Vendor ), "Quantity Received" and "Price/Unit".
  4. Select the values for "Type"(of Bags) and "VAT"( "None" if no VAT ) and "Discount"( "None" if no Discount ).
    The "Basic Amount" will be calculated based on the "Quantity Received" and "VAT" and displayed in the appropriate box given.
    The "Total Price" will be calculated automatically based on the "Discount" and the "Basic Price".
  5. You can select "Warehouse".

    6. Suppose to enter a purchase with Category "Birds", Code is "BRD101", Description is "BIRDS", Units is "Numbers", Quantity Sent is "100", Qty Received" is "99", Price/Unit is "12", VAT is "None", Discount is "None" and Warehouse is "GODOWN".
    You have to select Birds for "Category", BRD101 for "Code",None for "VAT", None for "Discount" and GODOWN for "Warehouse".
    As and when the "Code" value is selected the BIRDS for "Description" and Numbers for "Units" will be displayed automatically.
    Now enter 100 for "Qty Sent", 99 for "Qty Received". Note that now the Basic Amount is 1200.
  6. Repeat the steps 2 to 6 until the information of all Purchased items is filled.
  7. Enter values for "Vehicle No.", "Driver Name".

  8. Select value for "Frieght" as Included( in the Price/Unit) or Excluded or Excluded Paid By Supplier, Account Code and enter "Frieght Amount".
    Based on "Total Price" and "Frieght Amount" the value of "Grand Total" will be calculated.
  9. You have to select the Payment Mode for Frieght from dropdown named "Via" such as CASH, CHEQUE or OTHERS.
    If it is CASH you have to select "Cash Code" and value for "Dated"
    If it is CHEQUE you have to select values for "Bank A/C No", "Dated" and enter value for "Cheque"(Cheque Number).

    10. Suppose Frieght is "Included", Frieght Amount is "30", A/C code is "IE108", Payment is Via "Cash", Cash Code is "CIH", Date is "06.01.2011" then
    you have to select Included for "Frieght", IE108 for "A/C Code", Cash for "Via", CIH for "Cash Code", 06.01.2011 for "Dated" and enter 30 for "Frieght Amount". The "grand Total" now is 1170.
  10. Once filled click on 'SAVE' button to save the Direct Purchase or 'CANCEL' to exit without saving.

Add New Payment

  1. Select the "Date" of Payment, "Vendor", "Payment Method" and "TDS".
  2. selected value of "Payment Method"
  3. Select values for "Payment Mode","Unit".
  4. If the selected value for "Payment Mode" is CASH
      You have to select the value for "Cash Code".
      As and when the Cash Code value is selected the "Code" and "Description" values will be displayed automatically.
      Now you have to enter the value of "Amount".
    OR
    If the selected value for "Payment Mode" is CHEQUE
      You have to select the value for "Bank A/C No".
      As and when the Bank A/C No. value is selected the "Code" and "Description" values will be displayed automatically.
      Now you have to enter the value of "Amount", "Cheque #" and "Cheque Date".
  5. Now enter "Amount", "Cheque #" and "Cheque Date".
  6. If the selected value of "TDS" is WITH TDS,
      You have to select the value for "Code" and enter the value for "Amount" until the values for all the Payments are filled.

    7. Eg :Suppose the payment Date is "06 Jan, 2011", Vendor is "Obafemi", Payment Method is "Payment", TDS is "Without TDS", Unit is "Bangalore Godown", Payment Mode is "Cash", Cash Code is "CIH", Code is "AS111", Description is "1.CASH", Amount is "5000" SOBI is "SOBI-0111-0020" and Amount Paid is "4000",
    you have to select 06.01.2011 for "Date", Obafemi for "Vendor", Payment for "Payment Method", Without TDS for "TDS", Bangalore Godown for "Unit", Cash for "Payment Mode", CIH for "Cash Code". As and when the Cash Code value is selected the value of "Code" will be displayed as AS111 and "Description" is displayed as 1.CASH.
    Now enter 4000 for "Amount", select SOBI-0111-0020 for "SOBI" and enter 4000 for "Amount Paid".
  7. Once filled click "PAY" button to Save the Payment or click "CANCEL" button to exit without Saving the Payment.

Add New Credit Note

  1. You have to select the values of "Date", "Vendor","Unit" and enter "Amount"(Credit Amount = Total of Dr. - Total of Cr.).

    Eg :Suppose Date is "31 Dec, 2010" , Vendor is "Obafemi Trading Co.",Unit is "Bangalore Godown" and Amount is "100" you have to select 31.12.2010 for "Date", Obafemi Trading Co. for "Vendor",Bangalore Godown. for "Unit" and enter 100 for "Amount".
  2. Now select the values for "Code", "Cr/Dr" and enter the value of "Cr" or "Dr" depending on the select value of "Cr/Dr".
  3. Repeat the above step any number of times you want(Until all the entries are filled).

    4. Eg: If suppose you want to enter 2 entries with code "00104" with Cr value of "400" and Code "00502" with Dr value of "500" respectively ,
    You have to select 00104 for "Code", Cr for "Dr/Cr" and 400 for "Cr" for the first entry and
    select 00502 for "Code", Dr for "Dr/Cr" and 500 for "Dr" for second entry.
  4. You can enter "Narration" in the Textbox provided.
  5. Once filled click on 'SAVE' button to save the Credit Note or 'CANCEL' to exit without saving.
NOTE : You can Save the Credit Note only when the value of Total Of Dr. - Total of Cr. = Amount .
