Add New Place

Pre-requisites : "Supervisor" should be created.

  1. Click on "New Place".
  2. You have to select "Supervisor".
  3. Now enter the value for "Place(name)".

    Eg :Suppose the value of Supervisor is "Obafemi" and Place is "Lagos" then
    You have to select Obafemi for "Supervisor" and Lagos for "Place".


  4. Once filled click "SAVE" button to Save the Place or click "CANCEL" button to exit without Saving the Place.

Add New Farm

Pre-requisites : "Place" should be created.

  1. Click on "New Farm ".
  2. You have to select the values for "Supervisor", "Place"
  3. Now enter the values for "Farm Name", "Farmer Name", "Phone No. ", "Mobile No." and "Address".

    Eg :Suppose the Supervisor Name is "Obafemi", Place is "Lagos", Farm Name is "Obafemi PF", Farmer Name is "Obafemi", Phone No. is "080-41274443", Mobile No. is "9686567656" and Address is "Lagos" then
    You have to select Obafemi for "Supervisor", Lagos for "Place".
    Now enter Obafemi PF for "Farm Name", Obafemi for "Farmer Name", 080-41274443 for "Phone No.",9686567656 for "Mobile No." and Lagos for "Address".


  4. Once filled click "SAVE" button to Save the Farm or click "CANCEL" button to exit without Saving the Farm.

Add New Feed Cost Setings

  1. Click on "Feed Cost Setings".
  2. You have to enter the values for "From Date", "To Date", "Feed Cost/Kg", "Chick Cost/Bird", "Management Cost/Kg" and "Overhead Cost/Bird".

    Eg :Suppose the value of From Date is "1st Jul, 2011", To Date is "30th Aug, 2011", Feed Cost/Kg is "18", Chick Cost/Bird is "20", Management Cost/Kg is "1" and Overhead Cost/Bird is "1"
    you have to enter 01.07.2011 for "From Date", 30.08.2011 for "To Date", 18 for "Feed Cost/Kg", 20 for "Chick Cost/Bird", 1 for "Management Cost/Kg" and 1 for "Overhead Cost/Bird".
  3. Once filled click "SAVE" button to Save the Feed Cost Setings or click "CANCEL" button to exit without Saving the Feed Cost Setings.

Quail Daily Entry
  1. Select value for Supervisior.
  2. Select Farm, Feed Type,Medicine, Vaccine enter values for Flock, Age, Date(of entry), Mort(Mortality), Cull, Feed(In Kgs), Avg Wgt.(In Gms), Water, (Medicine)Quantity and ( Vaccine)Quantity.


    3. Eg : Suppose Supervisior is Obafemi, Farm is "Obafemi PF", Flock is "1", Age is "28", Date is "28 Feb,2011", Morality is "2", Feed Type is "FD111", Feed (In Kgs) is "120", Avg wt.(gms) is "1150", Vaccine is "VAC103" and Quantity is "2500" then
    You have to select Obafemi for "Supervisior", Obafemi PF for "Farm", 28.02.2011 for "Date", FD111 for "Feed Type", VAC103 for "Vaccine", enter 1 for "Flockno", 28 for "Age", 2 for "Mortality", 120 for "Feed (In Kgs), 1150 for "Avg wt.(gms)" and 2500 for "(Vaccine)Quantity".

Add New Quail Finalization

  1. Select values for Date, Supervisor, Farm and Flock.
  2. Now enter value for T.Cost(Transportation Cost) in appropriate input boxes provided.


    3. Eg : Suppose if the date is "10 June,2011", Supervisor is "Ehigha", Farm is "Healthy Farm", Flock is "MLR-111" and Transportation Cost is "1000" then
    You have select 10.06.2011 for "Date", Ehigha for "Supervisor", Healthy Farm for "Farm", MLR-111 for "Flock" and enter 1000 for "T.Cost".


  3. Now Click on "Save" button to save the Quail Finalization Details or "Cancel" to exit with out saving the Quail Finalization Details.

Add New Sale of Birds

  1. Select Values for "(Sale)Date" and enter value for "Book Invoice"(if any).
  2. Now select values for "Farm", "Customer(Name)".
  3. Enter values for "Birds(Sold)", "F.Weight(Farm Weight)", "Price/Kg", DC#, "Vehicle No.", select "Driver(Name)"(if any) and enter "remarks".


    4. Eg : Suppose if the Sale Date is "14th June, 2011", Book Invoice is "2011-14", Customer Name is "Ehigha", Farm Name is "Healthy Farm", No Of Birds sold were "1765", F.Weight is "3475.9", Price/Kg is "50", DC# is "3125", Vehicle No. id "KA14 C 1456" and driver Name is "Obafemi" then
    You have to select 14.06.2011 for "Date", Ehigha for "Customer", Healthy Farm for "Farm", Obafemi for "Driver".
    Now enter 1765 for "Birds", 3475.9 for "F.Weight", 50 for "Price/Kg", 3125 for "DC#" and KA14 C 1456 for "Vehicle No.".
  4. Repeat the above step 2 until all the sales details are filled.


  5. Now Click on "Save" button to save the Sale of Birds or "Cancel" to exit with out saving the Sale of Birds.

 Add New Birds Transfer

To Add a New Birds Transfer

  1. Select Values for "(Transfer)Date", "From Farm" , "From Flock", "To Farm", "To Flock", "Male Birds", "Female Birds" and "Price".

    2. Eg : Suppose if you want to transfer 500 male birds and 1000 female birds from ffarm "quail farm one", from flock "QuailF1" to to farm "test2", to flock "testut2-unit2-flock2" "4th June, 2012" then
    You have to select 04.06.2012 for "Date", quail farm one for "Farm",QuailF1 for "Flock",test2 for "To Farm", testut2-unit2-flock2 for "ToFlock", 500 for "Male Birds", 1000 for "Female Birds" and 25 as price.
  2. Repeat the step one to add multiple entries.
  3. Now Click on "Save" button to save the Transfer of Birds or "Cancel" to exit with out saving the Transfer of Birds.

 Fattener Birds Transfer

Add Fattener Birds Transfer

  1. You have to select date.
  2. You have to Enter DcNumber.
    Eg :Suppose the DcNumber is "1562" you have to enter 1562 for "DcNo".
  3. You have to select Processing Unit.
    Eg :Suppose the Processing Unit is "shopS57" you have to select processing unit for "Processing Unit".
  4. You have to enter Driver and Vehicle Number.
    Eg :Suppose the Driver is "Karan" and Vehicle Number is "KA07HR2331" you have to select Karan for "Driver" and KA07HR2331 for "Vehicle Number".
  5. You have to select Fattener Farm.
    Eg :Suppose the Fattener Farm is "farm" you have to select farm for "Fattener Farm".
  6. You have to select Flock.
    Eg :Suppose the Flock is "MLR-19" you have to select MLR-19 for "Flock".
  7. You have to enter number of birds.
    Eg :Suppose if birds are "200" you have to enter 200 for "Birds".
  8. You have to enter weight.
    Eg :Suppose if weight is "400" you have to enter 400 for "Weight".
  9. You have to enter Price.
    Eg :Suppose if price is "1250.50" you have to enter 1250.50 for "Price".
  10. You have to enter Mortality.
    Eg :Suppose if price is "10" you have to enter 10 for "Mortality".
  11. If any remarks write in remarks textbox
  12. Once filled click "SAVE" button to Save the Birds Transfer or click "CANCEL" button to exit without Saving the Birds Transfer.
