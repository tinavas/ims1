 Farm Details Report

Top

     You have to select Farm Details in Reports.
     As and when you select Farm Details, a new window will be opened.
     The Report shows farm code, Description,Capacity of farms fields and its values.

 House Details Report

Top

      You have to select House Details in Reports.
      As and when you select House Details, a new window will be open.
     The Report shows House Code,Description,Type,Capacity, Farm Code and Farm Description.

 Flock Details Report

Top

     You have to select Flock Details in Reports.
      As and when you select Flock Details, a new window will be open.
      The Report shows farm code, House code, Flock code, FlockDecription, Start date,Age,Female opening Fields and its values.

  1. You can select individual or all farm codes at a time using the FarmCode drop down list box.
  2. You can select single or all house codes at a time using the ShedCode

  3. you can also select Start date from which date you want to see the report, using the Startdate drop down list box.
      4. Depending on the options you select, report will diisplay Corresponding Values.
