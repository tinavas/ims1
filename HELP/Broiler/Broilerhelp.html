  1. Click on "Feed Cost Setings".
  2. You have to enter the values for "From Date", "To Date", "Feed Cost/Kg", "Chick Cost/Bird", "Management Cost/Kg" and "Overhead Cost/Bird".

    Eg :Suppose the value of From Date is "1st Jul, 2011", To Date is "30th Aug, 2011", Feed Cost/Kg is "18", Chick Cost/Bird is "20", Management Cost/Kg is "1" and Overhead Cost/Bird is "1"
    you have to enter 01.07.2011 for "From Date", 30.08.2011 for "To Date", 18 for "Feed Cost/Kg", 20 for "Chick Cost/Bird", 1 for "Management Cost/Kg" and 1 for "Overhead Cost/Bird".
  3. Once filled click "SAVE" button to Save the Feed Cost Setings or click "CANCEL" button to exit without Saving the Feed Cost Setings.

Broiler Daily Entry
  1. Select value for Supervisior.
  2. Select Farm, Feed Type,Medicine, Vaccine enter values for Flock, Age, Date(of entry), Mort(Mortality), Cull, Feed(In Kgs), Avg Wgt.(In Gms), Water, (Medicine)Quantity and ( Vaccine)Quantity.


    3. Eg : Suppose Supervisior is Obafemi, Farm is "Obafemi PF", Flock is "1", Age is "28", Date is "28 Feb,2011", Morality is "2", Feed Type is "FD111", Feed (In Kgs) is "120", Avg wt.(gms) is "1150", Vaccine is "VAC103" and Quantity is "2500" then
    You have to select Obafemi for "Supervisior", Obafemi PF for "Farm", 28.02.2011 for "Date", FD111 for "Feed Type", VAC103 for "Vaccine", enter 1 for "Flockno", 28 for "Age", 2 for "Mortality", 120 for "Feed (In Kgs), 1150 for "Avg wt.(gms)" and 2500 for "(Vaccine)Quantity".

  3. Now Click on "Save" button to save the Broiler Daily Entry or "Cancel" to exit with out saving the Broiler Daily Entry.

Add New Sale of Birds

  1. Select Values for "(Sale)Date" and enter value for "Book Invoice"(if any).
  2. Now select values for "Farm", "Customer(Name)".
  3. Enter values for "Birds(Sold)", "F.Weight(Farm Weight)", "Price/Kg", DC#, "Vehicle No.", select "Driver(Name)"(if any) and enter "remarks".


    4. Eg : Suppose if the Sale Date is "14th June, 2011", Book Invoice is "2011-14", Customer Name is "Ehigha", Farm Name is "Healthy Farm", No Of Birds sold were "1765", F.Weight is "3475.9", Price/Kg is "50", DC# is "3125", Vehicle No. id "KA14 C 1456" and driver Name is "Obafemi" then
    You have to select 14.06.2011 for "Date", Ehigha for "Customer", Healthy Farm for "Farm", Obafemi for "Driver".
    Now enter 1765 for "Birds", 3475.9 for "F.Weight", 50 for "Price/Kg", 3125 for "DC#" and KA14 C 1456 for "Vehicle No.".
  4. Repeat the above step 2 until all the sales details are filled.


  5. Now Click on "Save" button to save the Sale of Birds or "Cancel" to exit with out saving the Sale of Birds.
