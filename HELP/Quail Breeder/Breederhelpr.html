 Farm Details Report

Top

      You have to select Farm Details in Reports.
      As and when you select Farm Details, a new window will be opened.
      The Report shows farm code, Description, and Capacity of farms.

 Unit Details Report

Top

     You have to select Unit Details in Reports.
      As and when you select Unit Details, a new window will be opened.
     The Report shows farm code, farm Description, Unit Code,Unit Decription,OHT Capacity,Sump Capacity,Labour fields and its values.

  1. You can select single or all farms at a time using Farm code drop down list box.
      2. Based on the selected Farm corresponding details will be shown to you.

 Shed Details Report
