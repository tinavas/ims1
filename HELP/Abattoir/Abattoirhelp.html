Masters

   Add Chicken Rates Record

Top

    To Add Chicken Rates Record

  1. You have to select "Date" and "unit".


  2. Now select "Category" , "Code" and " Description"
    If you select the 'Code' you will get the 'Description' and if you select the 'Description' you will get the 'code'

    3. Eg : Suppose the Date is "6th June, 2011",unit is " Godown",Category "Chicken", Code "CHICK101" , Description "Boneless Chicken", Price "23".

    you have to select 06.06.2011 for "Date",Godown for "Unit ", Chicken for "Category " CHICK101 for "Code",Boneless Chicken for "Description",enter 23 for "Price".

  3. Repeat the above 2nd step until all the details are filled.


  4. Once filled click on 'SAVE' button to save the Chicken rate details or 'CANCEL' to exit without saving the Chicken Rate details.

    To Add Retail Rates Record

  1. You have to select "Date" , "Cash Code" and "Unit".


  2. Now select "Category" , "Code" and " Description"
    If you select the 'Code' you will get the 'Description' and if you select the 'Description' you will get the 'code'

  3. Enter the values for " Quantity " and " Price "

    4. Eg : Suppose the Date is "30th September, 2011",cashcode is "AS103",unit is " Godown",Category "Chicken", Code "CHICK101" , Description "Boneless Chicken", Quantity "35",Price "23".

    you have to select 30.10.2011 for "Date",AS103 for "Cashcode ",Godown for "Unit ", Hatch Eggs for "Category " CHICK101 for "Code",Boneless Chicken for "Description",enter 35 for "Quantity" and 23 for "Price".

  4. Repeat the above 2nd and 3rd steps until all the details are filled.


  5. Enter the Discount value and Cleaning Charges, then the Net Amount will be displayed automatically.
  6. Once filled click on 'SAVE' button to save the Chicken Retail Sales details or 'CANCEL' to exit without saving the Chicken Retail Sales details.

 Add New Live Birds Sale

Top

To Add a New Live Birds Sale

  1. Select Values for "(Sale)Date" and enter value for "Book Invoice"(if any),select values for "Customer(Name)" and Enter values for "Unit".
    2. Eg : Suppose if the Sale Date is "29th February, 2012", Book Invoice is "123", Customer Name is "RAHUL" and Unit Name is "Chicken Shop" then
    You have to select 29.02.2012 for "Date",123 for "Book Invoice", RAHUL for "Customer" and Chicken Shop for "Unit".
  2. You have to select code,description, "Birds(Sold)", "Weight", "Price/Kg", and enter "mortality".
    3. Eg : Suppose if the code is "BROB101", Birds are "100", Weight is "235.5", Price is "45" and Mortality is "5"
    You have to select BROB101 for "Code",100 for "Birds", 235.5 for "weight",45 for "Price",5 for "Mortality".
  3. Repeat the above step 2 until all the sales details are filled.
  4. Enter the Vehicle No and select Driver and enter discount amount if discount is given.
    5. Eg : Suppose if the Vehicle No. is "KA01KP143", Driver is "Trimurthy" and discount "10" then
    You have to enter KA01KP123 for "Vehicle No.",Trimurthy for "Driver" and 10 for "Discount".


  5. Now Click on "Save" button to save the Sale of Live Birds or "Cancel" to exit with out saving the Sale of Live Birds.

    To Add Chicken Processing Record

  1. You have to select "Date" and "unit".


  2. Select "From Type"," From Description" and "FromCode".
    If you select the 'Code' you will get the 'Description' and if you select the 'Description' you will get the 'code'


  3. Enter value for No of birds.


  4. Now select "Category" , "Code" and " Description"
    If you select the 'Code' you will get the 'Description' and if you select the 'Description' you will get the 'code'

    5. Eg : Suppose the Date is "6th June, 2011",unit is " Godown",FromType "Layer Birds",From Description "Layer Birds",From COde "LBRD101",No Of Birds "12",Category "Chicken", Description "BoneLess Chicken", Code "CHICK101" , Units "23",Quantity "31".

    you have to select 06.06.2011 for "Date",Godown for "Unit ", Layer Birds for "From Type ",Layer Birds for "FromDescription", LBRD101 for "FromCode",enter 12 for "No of birds", Chicken for "Category ",BoneLess Chicken for "Description" CHICK101 for "Code",enter 23 for "units",31 for "quantity".

  5. Repeat the above 4th step until all the details are filled.


  6. Once filled click on 'SAVE' button to save the Chicken Transfer Details or 'CANCEL' to exit without saving the Chicken Transfer details.

Add Broiler Birds Receiving

  1. You have to select date.
  2. You have to select ReceivedBy.
    Eg :Suppose if birds transfer is received by "steven" you have to select steven for "ReceivedBy".
  3. You have to select Chicken Shop(from where you received birds).
    Eg :Suppose if the Chicken Shop is "Puttur" you have to select Puttur for "Chicken Shop".
  4. You have to select DcNumber.
    Eg :Suppose the DcNumber is "1562" you have to select 1562 for "DcNo". After selecting DcNo total no of "birds sent" will be idsplayed automatically.
  5. You have to enter Birds Mortality.
    Eg :Suppose birds Mortality is "20" you have to enter 20 for "Birds Mortality". After entering birds mortality "Birds Remaining" will be displayed automatically.
  6. You have to enter Birds Received.
    Eg :Suppose birds Received are "600" you have to enter 600 for "Birds Received". After entering birds Received "Shortage" will be displayed automatically.
  7. Once filled click "SAVE" button to Save the Birds Receiving or click "CANCEL" button to exit without Saving Birds Receiving.
