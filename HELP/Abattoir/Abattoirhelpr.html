 Category Wise Production

Top

      You have to select Category Wise Production in Reports.
      As and when you select Farm Details, a new window will be opened.

  1. Select Unit, from the drop down list.
  2. Select Flock, from the drop down list.
  3. Select From Date and To Date.
  4. Click on Report Button.
      5. Report will be displayed in a new wondow.

 Chicken Stock Ledger

Top
