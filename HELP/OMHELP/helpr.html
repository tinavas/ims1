 Call Register

Top

     You have to select Call Register in Reports.
     As and when you select Call Register, a new window will be opened.
     The Report shows Employee, Time From, To Time, Dialed and Narration.

 Client Appointment

Top

     You have to select Client Appointment in Reports.
     As and when you select Client Appointment, a new window will be opened.
     The Report shows Clients Entry Date, Client Name, Employee,Appointment Time From,Appointment Time to and Narration fields and its values.

  1. You can select single or all Employees at a time using Employee drop down list box.
      2. Based on the selected Employee corresponding details will be shown to you.

 Client Assessment Report

Top

     You have to select Client Assessment in Reports.
     As and when you select Client Assessment, a new window will be opened.
     The Report shows Client Name, Type, Employee,Date From and Date To fields and its values.

  1. You can select single or all Client's Names at a time using Client Name drop down list box.
      2. Based on the selected Client Name corresponding details will be shown to you.

 Client History Report

Top

     You have to select Client History in Reports.
     As and when you select Client History, a new window will be opened.
     The Report shows Client, Employee, Interaction Type,Document Number and Narration fields and its values.

 Client Ledger Report

Top

     You have to select Client Ledger in Reports.
     Select Client, From Date, To Date and Client on Report.      As and when you click report, a new window will be opened.
     The Report shows Date, Transaction Type,Document Number,Debit and Credit fields and its values.

 Client List Report

Top

     You have to select Client List in Reports.
     As and when you select Client List, a new window will be opened.
     The Report shows Client Name, Place, Mobile,Land Line,Email,Unique Id,TIN,TAN,Category and Ward fields and its values.

 Tax Filing Report

  1. You can select single or all Employees at a time using Employee drop down list box.
