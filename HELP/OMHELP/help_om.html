Masters


   Add Client

Top

    To Add New Client Master

  1. You have to enter Name.
    2. Eg : Suppose Client name is "Steven".
    you have to enter Steven for "Client Name".
  2. You have to enter Surname.
  3. You have to enter Father's Name.
    4. Eg : Suppose Fathers name is "Johnson".
    you have to enter Johnson for "Fathers Name".
  4. You have to select Sex.
  5. You have to enter Office Address.
    6. Eg : Suppose Office address is "Tulasi technologies, Bangalore".
    you have to enter Tulasi Technologies, Bangalore for "Office Address".
  6. You have to enter Office Place(location of your office).
    7. Eg : Suppose Office Place is "Bangalore".
    you have to enter Bangalore for "Office Place".
  7. you have to enter Residential Address & residential place.
  8. You have to enter Annual Income.
    9. Eg : Suppose Annual Income is "50000" .
    you have to enter 50000 for "Annual Income".
  9. You have to select Category.
    10. Eg : Suppose Category is "CAT01".
    you have to select CAT01 for "Categorye".
  10. You have to select Ward.
    11. Eg : Suppose Ward is "WARD01".
    you have to select WARD01 for "Ward".
  11. You have to enter landline Number & Mobile Number.
  12. You have to enter E-mail ID.
    13. Eg : Suppose your E-mail id is "steven@gmail.com".
    you have to enter steven@gmail.com for "E-mail".
  13. You have to enter TIN(Tax Information Network),TAN(Tax deduction Account Number) and UID(Unique Identity Number).
  14. You have to Select Client Group.
    15. Eg : Suppose if Client Group is "C101".
    you have to selectC101 for "Client Group".
  15. Once filled click on 'save' button to save details or 'CANCEL' to exit without saving the details

 To Add Client Appointment Details

Top

  1. You have to select date, on which date appointment is fixed.
    Eg :Suppose if Date is "05.09.2011" then, you have to select 05.09.2011 as "Date".
  2. You have to select employee to whom appointment is to be assign.
    Eg :Suppose if you want to assign to "Steven" you have to select Steven as "Employee".
  3. You have to select Unique Number it may be PAN(Perminent Account Number) or PASSPORT Number.
    Eg :Suppose if your U.No is "UID4537" then you have to select UID4537 as "U.No".
  4. You have to select Appointment Date.
    Eg :Suppose if Appointment Date is "25.09.2011" then you have to select 25.09.2011 as "Appointment Date".
  5. You have to enter Time From and Time To.
    Eg :Suppose if Appointment is from "08:30" to "11:30" then you have to enter 08:30 as "Time From" and 11:30 as "Time To".
  6. If you want to write something about Appointment, write it in narration box.

    7. Once filled click "SAVE" button to Save Client Appointment or click "CANCEL" button to exit without Saving Client Appointment.

Top

  2. You have to select employee who receive or make the call
    Eg :Suppose if "Steven" answer that call, you have to select Steven as "Employee".

    Eg :Suppose if U.No is "UID4537" then you have to select UID4537 as "U.No".

    Eg :Suppose if number is "9656764656" you have to enter 9656764656 as "dialled/out no".

    You have to enter call StartTime and End Time.
    Eg :Suppose if call duration is from "08:30:45" to "08:36:25" then you have to enter 08:30:45 as "Start Time" and 08:36:25 as "End Time".
  6. If you want to write something about Call, write it in narration box.
  7. Once filled click "SAVE" button to Save Call Register or click "CANCEL" button to exit without Saving Call Register.

 Receipt

Top

  1. You have to select "Date".
    Eg :Suppose date is "25.09.2011" you have to select 25.09.2011 for "Date".
  2. You have to select Unique Number it may be PAN(Perminent Account Number) or PASSPORT Number.
    Eg :Suppose if U.No is "UID4537" then you have to select UID4537 as "U.No".
  3. You have to select "Choice".
    Eg :Suppose if you paying amount On Account then you have to select On A/C for "choice" (or) if you paying for individual invoices then you have to select Invoice as "choice".
  4. You have to select "Reception Mode".
    Eg :Suppose if amount received by cheque then you have to select Cheque as "Reception Mode". (or) If you received cash then you have to select Cash as "Reception Mode".
  5. You have to select cash code or bank account number(based on the option you selected for Reception Mode, it will automatically give the options regarding it).
    Eg :Suppose if Cash Code is "CH" then you have to select CH as "Cash Code".
  6. If you selected choice as "On A/C". then you have to enter Amount,Bank,Branch,Cheque Number,Cheque Date(if Reception mode is Cheque then only cheque number and cheque date is required).
    Eg :Suppose if Amount is "10000" then you have to select 10000 as "Amount",Bank is "ICICI" then you have to select ICICI as "Bank", Branch is "Banglore" then you have to select Banglore as "Branch",cheque number is "956847659" then you have to select 956847659 as "Cheque" and cheque date is "19.09.2011" then you have to select 19.09.2011 as "Cheque Date" .
  7. If you selected Choice as "Invoice", then the list of invoices will displayed of particular customer. You have to enter received amount for each invoice.
  8. If you want to write something about Reciept, write it in narration box.
  9. Once filled click "SAVE" button to Save Receipt or click "CANCEL" button to exit without Saving Receipt.
