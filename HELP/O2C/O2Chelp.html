    you have to enter Obafemi P/F for "Name", Nigeria for "Address", 23414228040 for "Phone", 9141414223 for "Mobile" and 10,000 for "Credit Limit".
    You have to select Customer&Supplier for "Contact Type" and 47778 for "Customer Group" and 2 days for "Credit Term".

   Add New Tax Master

    To Add new Tax Master
    Top
  1. You have to enter the values of "Tax Code" and "Description."
  2. You have to select the value of "Deductions" , if it is "Charges" you have to selct the value of "Charge Type" also.
  3. Now select the value for "CoA".
  4. Select either "Percentage" or "Flat" by selecting the appropriate radio button and enter the values of "Tax Code Value".
  5. You have to select the "Applicable" weather taxcode is applicable only for sales or it has to applicable for both purchases and sales.
    6. Eg :Suppose Tax Code is "tax001", Description is "taxcode001", Deductions is "Charges", Charge Type is "Brokerage" CoA is "IT401", Tax Code Value is "1.5" and Applicable is "sales",
    you have to enter tax001 for "Tax Code", taxcode001 for "Description", select Charges for "Deductions", Brokerage for "Charge Type", IT401 for "CoA", select Sales for "Applicable."
    Now enter 1.5 for "Tax Code Value".
  6. Once filled click on 'SAVE' button to save the Tax Master or 'CANCEL' to exit without saving.

   Add New Direct Sale

Top

    To Add new Direct Sale
  1. You have to select the "Date" of the Sales, "Customer", and enter the value of "Book Invoice"(if not available make it "0"(Zero) ) .
    The "Invoice" value will be generated automatically based on which Receipts will be made.

    2. Eg: If suppose for the Invoice "COBI-0111-0020", Sales Date is "05.01.2011" and Customer is "Adeniji" you have to select 05.01.2011 for "Date" and Adeniji for "Customer".
    Note that the Invoice Number COBI-0111-0020 will be generated automatically.
  2. You can select "Category" and "Code".
    As and when the "Code" value is selected the "Description" and "Units" of the selected item will be displayed automatically.
  3. Now enter the "Quantity" and "Price/Unit".
  4. Now select the "VAT" and "Discount".
    The "Basic Amount" will be calculated based on the "Quantity Received" displayed in the appropriate box given.

    5. Suppose to enter a Sale with Category "Birds", Code is "BRD101", Description is "BIRDS", Units is "Numbers", Quantity is "100" and Price/Unit is "12",VAT is "TAX001" and "Discount"is "Disc001"
    You have to select Birds for "Category", BRD101 for "Code"
    As and when the "Code" value is selected the BIRDS for "Description" and Numbers for "Units" will be displayed automatically.
    Now enter 100 for "Qty" and 12 for "Price/Unit". You have to select TAX001 for "VAT", DISC001 for "Discount"
    Note that now the Basic Amount is 1200.
  5. Repeat the steps 2 and 3 until the information of all sale items is filled.
  6. The "Total Price" will be calculated automatically based on the "Discount" and the "Basic Price".

    7. Suppose Discount Amount is "38" you have to select Amt radio button and enter 38 for "Amount". Now the Total Price is displayed as 1162.
  7. Enter values for "Vehicle No.", "Driver Name" and select appropriate value for "Broker Name" from the dropdown provided.

    8. Let Broker Name is "Adeniji", Vehicle Number is "AA79-AKK" and Driver Name is "Arikawe" then you have to enter AA79-AKK for "Vehicle No", Arikawe for "Driver Name" and select Adeniji for "Broker Name".
  8. Select value for "Freight" as Included( in the Price/Unit) or Excluded , Account Code and enter "Freight Amount".
    Based on "Total Price" and "Freight Amount" the value of "Grand Total" will be calculated.
  9. You have to select the Payment Mode for Freight from dropdown named "Via" such as CASH, CHEQUE or OTHERS.
    If it is CASH you have to select "Cash Code" and value for "Dated"
    If it is CHEQUE you have to select values for "Bank A/C No", "Dated" and enter value for "Cheque"(Cheque Number).

    10. Suppose Freight is "Included", Freight Amount is "30", A/C code is "IE108", Payment is Via "Cash", Cash Code is "CIH", Date is "06.01.2011" then
    you have to select Included for "Freight", IE108 for "A/C Code", Cash for "Via", CIH for "Cash Code", 06.01.2011 for "Dated" and enter 30 for "Freight Amount". The "grand Total" now is 1192.
  10. Once filled click on 'SAVE' button to save the Direct Sale or 'CANCEL' to exit without saving.

   Edit Existing Direct Sale

    To Edit Existing Direct Sale
    Top
  1. You have to select the "Date" of the Sales, "Customer", and enter the value of "Book Invoice"(if not available make it "0"(Zero) ) if you want to change the previous values.

    2. Eg: If suppose you want to change the Sales Date is from "1st Jan, 2011" to "5th Jan, 2011" you have to select 05.01.2011 for Date.
  2. You can select "Category" and "Code" if you want to change the previous values.
    As and when the "Code" value is selected the "Description" and "Units" of the selected item will be displayed automatically.
  3. Now enter the "Quantity" and "Price/Unit" if you want to change the previous values.
  4. Now select the "VAT" and "Discount" if you want to change the previous values.
    The "Basic Amount" will be calculated based on the "Quantity Received" and "VAT" and displayed in the appropriate box given.

  5. Repeat the steps 2 and 3 until the information of all sale items is filled.
    6. The "Total Price" will be calculated automatically based on the "Discount" and the "Basic Price".
  6. Select value for "Freight" as Included( in the Price/Unit) or Excluded , Account Code and enter "Freight Amount".
    Based on "Total Price" and "Freight Amount" the value of "Grand Total" will be calculated.
  7. You have to select the Payment Mode for Freight from dropdown named "Via" such as CASH, CHEQUE or OTHERS.
    If it is CASH you have to select "Cash Code" and value for "Dated"
    If it is CHEQUE you have to select values for "Bank A/C No", "Dated" and enter value for "Cheque"(Cheque Number).

  8. Once filled click on 'SAVE' button to save changes to the selected Direct Sale or 'CANCEL' to exit without saving the changes to the selected Direct Sales.

   Add New Payment

    To Add a New Payment

    Top
  1. Select the "Date" of Payment, "Customer", "Payment Method",unit.
  2. Select values for "TDS" and "Reception Mode".
  3. If the selected value for "Reception Mode" is CASH
      You have to select the value for "Cash Code".
      As and when the Cash Code value is selected the "Code" and "Description" values will be displayed automatically.
      Now you have to enter the value of "Amount".
    OR
    If the selected value for "Reception Mode" is CHEQUE
      You have to select the value for "Bank A/C No".
      As and when the Bank A/C No. value is selected the "Code" and "Description" values will be displayed automatically.
      Now you have to enter the value of "Amount", "Cheque #" and "Cheque Date".
    OR
    If the selected value for "Reception Mode" is OTHERS
      You have to enter the values for "Code" and "Description".
      As and when the Bank A/C No. value is selected the "Code" and "Description" values will be displayed automatically.
  4. If the selected value of "TDS" is WITH TDS,
      You have to select the value for "Code" and enter the value for "Amount" until the values for all the Payments are filled.
    5. Eg :Suppose the payment Date is "06 Jan, 2011", Customer is "Chiwetel",Unit is "BAngalore Godown", Payment Method is "Payment", TDS is "Without TDS", Choice is "On A/C", Reception Mode is "Cash", Cash Code is "CIH", Code is "AS111", Description is "1.CASH", Amount is "5000",
    you have to select 06.01.2011 for "Date", Chiwetel for "Customer", Bangalore Godown for "Unit", Payment for "Payment Method", Without TDS for "TDS", On A/C for "Choice", Cash for "Reception Mode", CIH for "Cash Code". As and when the Cash Code value is selected the value of "Code" will be displayed as AS111 and "Description" is displayed as 1.CASH.
    Now enter 4000 for "Amount".
  5. Once filled click "PAY" button to Save the Payment or click "CANCEL" button to exit without Saving the Payment.

  3. Repeat the above step any number of times you want(Until all the entries are filled).
    4. Eg: If suppose you want to enter 2 entries with code "00104" with Cr value of "400" and Code "00502" with Dr value of "500" respectively ,
    You have to select 00104 for "Code", Cr for "Dr/Cr" and 400 for "Cr" for the first entry and
    select 00502 for "Code", Dr for "Dr/Cr" and 500 for "Dr" for second entry.
  4. You can enter "Narration" in the Textbox provided.
  5. Once filled click on 'SAVE' button to save the Credit Note or 'CANCEL' to exit without saving.
NOTE : You can Save the Credit Note only when the value of Total Of Dr. - Total of Cr. = Amount .
